ElliptiKeys is a project made to educate people about the security of the Bitcoin network. Many people are concerned that their private key will be brute-forced, and this site is meant to show that this will simply never be the case.

All of the code for the site is 100% open-source and can be compiled and ran on any computer :)

If you wish to support this site, any donation is greatly appreciated!
