Source Code

ElliptiKeys Raw Angular Source Code

This is the full, uncompiled Angular source code for ElliptiKeys. Use this if you wish to make your own changes and host/publish them, or if you plan on self-compiling for secure wallet generation (Recommended if generating important wallets). Requires installing Angular CLI.

Precompiled Angular Build

This is the precompiled build of the site. Comes ready to host. Use this if you plan on hosting your own version locally or globally without making any changes. Requires HTTP server software, such as Live Server for Visual Studio Code.

ElliptiKeysJS Library

Easy-to-use JavaScript key generator that powers the website. Works directly in the browser. Use this if you would like a simple solution for generating Bitcoin keys and mnemonics in your web app.
