About Testnet & Testnet Scams

The Bitcoin Testnet is an alternative Bitcoin blockchain functionally identical to the original. It is intended for developer use, so that blockchain developers can test new updates without worrying about losing actual Bitcoins or accidentally breaking the main blockchain.

Due to this, testnet wallets and coins alike possess no monetary value, and are NEVER meant to be used for real-life payments or purchases.


How testnet scams work

The primary way a testnet scam gets executed is by someone convincing someone else to make a Bitcoin wallet to receive a payment, but instead tricking them into making a testnet wallet and sending worthless testnet coins and not actual Bitcoin.

How to avoid testnet scams

The easiest way to tell if you are getting scammed is by looking at the address prefix.

  • A real Bitcoin address should always start with 1, 3, or bc1q/bc1p
  • A worthless testnet address will be prefixed with m, n, 2, or tb1q/tb1p. If you have been asked to create a wallet that starts with one of these, you are being scammed.
