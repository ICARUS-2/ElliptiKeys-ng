About ElliptiKeys

As you could probably tell, this site is based off the ever-popular keys.lol site. However, this version is capable of deriving every public key type including SegWit and Bech32 addresses, as opposed to only Legacy-type addresses. This version also displays all compressed private keys alongside uncompressed ones.

It's also in dark mode.

How it works

Knowing how massive the possible keyspace truly is, storing every single private and public key simply would not be possible. The keys are all dynamically computed when the page is loaded.

Since we know a private key is simply a number somewhere between 1 and 2^256, it calculates them starting at n=([page number] * [keys/page]) and derives every private key and its corresponding address formats from n to n + [keys/page]. Once the private and public keys have been derived, it uses the Blockchain.com API to perform the balance checking.
